# movie_review_analysis
네이버 영화리뷰 분석 및 비교 


# Summary

## 1. mecab 와 sentencepiece 형태소 분석기를 사용하여 네이버 영화리뷰 내용을 분석

## 2. 각각의 모델 성능을 ROC 커브를 계산하여 비교
- ## mecab: area = 0.932
- ## sentencepiece area = 0.878
